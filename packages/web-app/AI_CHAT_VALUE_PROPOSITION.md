# AI Chat Value Proposition: Bio KGnome as Research Assistant

## Core Product Understanding
**Bio KGnome is NOT selling reports. It's selling:**
- **AI Chat Interface** - Like ChatGPT for biomedical research
- **Powered by Custom Knowledge Graph** - Built from 127+ biomedical APIs
- **Vector-Enabled** - Semantic search across biomedical data
- **Evidence-Backed Answers** - Every response cites exact sources
